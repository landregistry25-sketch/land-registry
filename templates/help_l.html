Land Inspector Dashboard - Document Verification Workflow

Reviewing Verification Requests

  1. Login to the Land Inspector portal and access the dashboard to see all incoming verification requests.
  2. Each request corresponds to a land/property and may have multiple documents uploaded for verification.
  3. The dashboard displays a Verify button per request; this button is enabled only if no documents are rejected.
  4. Click View Documents to open detailed document information for each land request.

Verifying and Rejecting Documents

  1. On the document details page, review each uploaded document carefully including Aadhar card, land map, sale deed, etc.
  2. Each document will have two buttons: Verify and Reject.
  3. Click Verify to mark a document as verified.
  4. Click Reject to mark a document as rejected. Provide a reason for rejection in the text area provided.
  5. If any single document is rejected, the overall Verify button on the dashboard will be disabled for that request.
  6. Users are notified about verification or rejection statuses of their documents and requests.

Finalizing the Verification Process

  1. Once all documents for a land request are verified, you can use the Verify button on the dashboard to approve the request.
  2. Rejected requests must be corrected by the user and resubmitted for new verification cycle.
  3. Maintain clear communication with users through notifications about the status of their documents and requests.
Always ensure to verify each document carefully to maintain the integrity of the land registry system.
